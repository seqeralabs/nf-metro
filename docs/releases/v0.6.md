## v0.6.0 — variantbenchmarking pipeline support

*2026-03-06* · [Release](https://github.com/pinin4fjords/nf-metro/releases/tag/0.6.0)

Layout and routing improvements driven by adding the [nf-core/variantbenchmarking](https://github.com/nf-core/variantbenchmarking) pipeline to the gallery. The pipeline has a complex fan-out/fan-in topology with multiple converging routes, which exposed several edge cases in the inter-section routing engine.
