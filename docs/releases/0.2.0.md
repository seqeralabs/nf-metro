# v0.2.0

*2026-02-17* · [GitHub release](https://github.com/pinin4fjords/nf-metro/releases/tag/0.2.0) · [Diff](https://github.com/pinin4fjords/nf-metro/compare/0.1.1...0.2.0)

## Section overlap fixes and layout engine hardening

Fixed section overlaps in the rnaseq layout and hardened the layout engine against a broader range of topologies, including a new suite of topology stress tests covering fan-out, fan-in, diamonds, folds, and mixed port sides.

## Docs site

Launched the [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) documentation site at [pinin4fjords.github.io/nf-metro](https://pinin4fjords.github.io/nf-metro/), with an example gallery and versioned docs via `mike` (separate `dev` and release-tagged deploys).

## Codebase refactor

Extracted constants, split modules, removed dead code, and improved the README with a full CLI reference.
